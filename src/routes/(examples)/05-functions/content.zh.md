# Functions

在Sway中，函数是执行操作的核心构建块，它们可以修改数据、执行计算，并返回结果。

1. **内部和外部函数**：
   - 内部函数（`fn`）：这些函数是在智能合约内部定义和使用的，不能从外部调用。
   - 外部函数（`abi`）：这些函数是智能合约的公共接口，可以被外部调用。
2. **`ref mut`**：
   - `ref mut`是Sway中的一个关键字，用于创建可变引用，允许函数修改传递给它的参数。
3. **返回多个输出**：
   - 函数可以返回多个值，这些值通常以元组的形式返回。
